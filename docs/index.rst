Boulder Examples
================

Catalog of `Cantera`_ reactor-network samples expressed as Boulder STONE YAML files.

Runnable examples include post-solve GUI screenshots with the **Plots** panel visible.
YAML files are generated with ``sim2stone`` from vendored upstream scripts (see
:doc:`contributing`).

.. _Cantera: https://github.com/Cantera/cantera/tree/main/samples/python/reactors

Launch in Codespaces
--------------------

Open `this repository in a Codespace`_ and run::

   boulder examples/cantera/<example>.yaml --no-open

.. _this repository in a Codespace: https://github.com/codespaces/new?hide_repo_select=true&repo=parks4/boulder_examples
